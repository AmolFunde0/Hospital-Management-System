# Hospital Management Frontend (React + Vite)

This React frontend connects to your Spring Boot backend APIs at `http://localhost:8080/api/v1`.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Open `http://localhost:5173` (or the URL shown by Vite) in your browser.

## Notes
- The app uses Bootstrap CDN for styling.
- Make sure your Spring Boot backend is running and CORS is enabled (or proxy).
- API endpoints used:
  - Patients: `/api/v1/patients`
  - Doctors: `/api/v1/doctor`
  - Appointments: `/api/v1/appointment`
  - Bills: `/api/v1/bill`
